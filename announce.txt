Subject: ffcall 2.1-1

The following packages have been uploaded to the Cygwin distribution:

* libffcall0-2.1-1
* libffcall-devel-2.1-1

libffcall is a library which can be used to build foreign function
call interfaces in embedded interpreters.

This is an update to the latest upstream release.  It is the first
release for 64-bit Cygwin.
